    Button (버튼)

    • Components
    • Action (액션)
    • Button (버튼)
    Design Code

    Button

    • 버튼은 어떤 기능이나 동작을 실행하거나 기능을 사용하기 위한 상태로 변경하는 요소이다.

    • 사용자가 서비스를 이용하는 과정에서 어떤 행동이 중요한지에 따라 관련된 버튼이 다양한 스타일로 표현된다.

    구조

    • 1텍스트 레이블 :

      버튼을 눌렀을 때 실행되는 기능에 대한 설명

    • 2컨테이너(선택) :

      버튼을 배경과 구분하는 시각적인 수단으로 면 또는 선으로 표현됨

    • 3아이콘(선택) :

      버튼을 눌렀을 때 실행되는 기능을 텍스트 대신 시각적인 상징으로 표현한 것

    계층

    • Primary : 가장 중요한 명령을 실행 시킬 때 사용하다.

    • Secondary : Primary 버튼과 함께 사용할 수 있으며 시각적 주목도가 Primary 버튼보다 낮다.

    • Tertiary : 주로 판단에 영향을 주지 않는 경우 사용한다.

    Filled
    Outline
    Transparent

    상태

    • 1Default :

      기본

    • 2Hover:Active :

      호버/눌림/포커스

    • 3Disabled :

      비활성화

    조합

    아이콘 + 텍스트
    텍스트 + 아이콘
    아이콘 + 텍스트 + 아이콘
