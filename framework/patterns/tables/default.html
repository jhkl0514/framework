테이블_기본형

  • Patterns
  • Tables (테이블)
  • 기본형

기본형

• 기본형 테이블은 데이터나 정보를 구조적으로 정리해 놓은 표이다. 이 테이블은 주로 데이터를 시각적으로 명확하게 전달하기 위해 사용되며, 사용자가 직접 데이터를 입력하거나 수정하지 않는 형태로 구성되어 있다.

구조

a. 세로형_디폴트

• 헤더가 상단에 고정되고 하단에 여러 행으로 구성되어있는 가장 일반적인 테이블 형태

b. 세로형_스크롤

• 테이블 내용이 많아 정해진 Width 값을 벗어난 경우 스크롤 처리를 하며, 중요도에 따라 일부 셀을 고정하고 스크롤 처리 할 수 있다.

c. 가로형

• 헤더가 왼쪽, 행이 오른쪽으로 구성되어 있는 테이블 형태

• 가로형 테이블은 스크롤 형태가 없다.

사용법

  • • 테이블 상단영역에는 테이블 탭 최대 5개, 테이블 버튼은 최대 4개까지만 노출한다.

  • 이로움케어 리뉴얼 전 가로사이즈(1060px)사이즈를 고려하여 테이블 탭, 버튼 개수를 제한한다.

  • 1테이블 탭
  • 2테이블 버튼
  • • 데이터 셀 안에 링크가 있을 경우 언더라인으로 표시한다.

  • 언더라인을 표시하여 링크임을 명확하게 인지할 수 있도록 하며, 웹 접근정 측면에서도 중요한 역할을 한다.
    링크는 팝업 또는 페이지로 이동한다.

  • • 테이블 섹션 타이틀의 경우에도 테이블 버튼은 최대 4개까지만 노출한다.

  • 버튼 개수는 최소한 적게 사용하는 것을 원칙으로 한다.

  • • 테이블 페이지네이션 조건은 데이터 개수가 20개를 초과하는 경우에만 노출한다.

  • 페이지네이션에 대한 자세한 내용은 페이지네이션 컴포넌트 참고 (페이지네이션 컴포넌트 바로가기)
