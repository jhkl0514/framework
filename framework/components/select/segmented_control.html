    Segmented Control (세그먼트 컨트롤)

    • Components
    • Select (선택)
    • Segmented Control (세그먼트 컨트롤)
    Design Code

    Segmented Control

    • 세그먼트 컨트롤은 사용자가 여러 옵션 중 하나를 선택할 수 있도록 제공한다.

    • 일반적으로 상호 배타적인 옵션을 표시하는 데 사용되며, 라디오 버튼이나 드롭다운 메뉴와 유사한 기능을 제공한다.

    구조

    • 1 레이블 :

      세그먼트와 연결된 대상을 안내하는 텍스트

    • 2선택된 세그먼트 :

      세그먼트 옵션 선택된 상태임을 알려주는 시각적인 식별자

    • 3컨테이너 :

      세그먼트를 영역을 면 또는 선으로 표현됨

    • 4구분 선 :

      세그먼트를구분하는 시각적인 수단으로 선으로 표현됨

    용도

    • • 설정 옵션 선택 :

      사용자가 설정 옵션 중 하나를 선택할 수 있도록 한다. (ex. 언어 설정, 음량 설정 등을 제어하는 데 사용)

    • • 필터링 :

      사용자가 데이터를 필터링할 수 있도록 한다. (ex. 카테고리별로 항목을 필터링하거나 날짜 범위별로 결과를 필터링하는 데 사용)

    • • 정렬 :

      사용자가 데이터를 정렬할 수 있도록 한다. (ex. 이름별로 항목을 정렬하거나 날짜별로 결과를 정렬하는 데 사용)

    가이드

    • 한 번에 하나의 제어 옵션만 선택하고 활성화할 수 있다.

    • 기본적으로 하나의 옵션이 선택되어 있어야 한다.

    • 라벨은 짧고 간결해야 하며, 레이블이 세그먼트 내에 맞추기에 너무 긴 경우 아이콘 전용 변형을 사용하는 것을 고려한다.

    • 사용자에게 제공할 옵션의 수가 너무 많으면 사용자가 선택하기 어려울 수 있다.

    • 화면 공간을 많이 차지하지 않도록 적절한 크기로 설정해야 한다.

    상태

    • • Square type :

      상태 값은 Default, Hover:Press, Active상태 값을 표현한다.

    • • Round type :

      상태 값은 Default, Active상태 값을 표현한다.

    사이즈&간격

    • 세그먼트 컨트롤의 너비는 텍스트 길에 따라 너비를 가변형진행하여 다양한 레아아웃에 대응한다.

    • 반응형 설계시 기본적으로 전체 너비를 100%로 진행한다.

//toggleWrap
$(".tableTabTopcustom").each(function () {
  const findSpan = $(this).find("span.active");
  const spanWidth = $(findSpan).innerWidth();
  const spanoffset = $(findSpan).position().left;
  $(this).find(".bar").css({ width: spanWidth, left: spanoffset });
  $(this)
    .find("span")
    .on("click", function () {
      const spanWidth = $(this).innerWidth();
      const spanoffset = $(this).position().left;
      $(this).parent(".tableTabTopcustom").find("span").removeClass("active");
      $(this)
        .parent()
        .find(".bar")
        .css({ width: spanWidth, left: spanoffset });
      $(this).addClass("active");
    });
});
