    Select (셀렉트)

    • Components
    • Select (선택)
    • Select (셀렉트)
    Design Code

    Select

    • 셀렉트는 사용자에게 여러 개의 옵션 목록을 팝업으로 제공하여 그 중 한 개의 값을 선택할 수 있도록 하는 경우에 사용함

    구조

    • 1레이블 :

      사용자가 옵션 목록의 카테고리에 대한 설명 또는 옵션 선택에 대한 도움말을 제공하는 문구

    • 2컨테이너 :

      셀렉트와 배경을 구분하는 시각적인 수단으로 면 또는 선으로 표현함

    • 3도움 텍스트 :

      입력 내용, 입력 방식에 대한 도움말 또는 오류 메시지를 제공함

    • 4플레이스홀더 :

      어떤 값을 선택해야 하는지에 대한 힌트 또는 예시를 제공함

    • 5선택값 :

      사용자가 옵션 목록에서 선택한 값을 보여줌

    • 5옵션 :

      사용자가 선택할 수 있는 값 또는 사용자에게 제안된 값의 목록

    상태

    • 1Inactive :

      선택되지 않은 선택

    • 2Default :

      입력이 완료된 상태

    • 3Focused :

      선택되거나 입력중인 상태

    • 4Disabled :

      비활성화

    • 5Error :

      에러

    • 6ReadOnly :

      사용자는 값을 볼 수만 있고, 수정이나 입력은 불가능한 상태

    스타일

    • 옵션 텍스트를 간결하게 제공한다.

    • 모든 셀렉트에는 레이블을 제공한다.

    • 셀렉트에 값을 변경하였을 때 폼이 제출되어서는 안 된다.

    사이즈&간격

    사용법

    • • 셀렉트 필드는 텍스트의 길이를 고려하여 적절한 크기로 제공한다.

    • 기본적으로 셀렉트 필드의 크기를 그리드, 화면 크기에 따라 유동적이나 전체 섹션 영역을 차지하도록 구현한다. 그러나 검색, 시간 등 입력값의 길이를 예측할 수 있는 경우 최대 길이에 맞추어 입력 필드의 너비를 고정하고, 사용자가 더 작은 크기의 화면에서도 잘림 없이 입력값을 확인할 수 있는지를 점검해야 한다.

    • • 모든 텍스트 입력 필드에는 레이블을 제공한다.

    • 셀렉트 필드에 레이블이 제공되지 않으면 사용자는 어떤 정보를 입력해야 하는지 알 수 없다. 레이블을 생략하고자 하는 경우에는 레이블 없이도 사용자가 값을 선택하는 데 문제가 없다는 근거가 명확해야 한다.

//selectBox jquery
$(".DDY_select .select").on("click", function () {
  const option = $(this).siblings(".option");
  $(this).toggleClass("active");
  option.toggleClass("active");
  if ($(this).hasClass("disable")) {
    $(this).removeClass("active");
    option.removeClass("active");
  }
  if ($(".DDY_select .select").not(this).hasClass("active")) {
    $(".DDY_select .select")
      .not(this)
      .removeClass("active")
      .siblings(".option")
      .removeClass("active");
  }
});

//option 클릭시 remove
$(".option button").on("click", function () {
  const text = $(this).text();
  $(this).closest(".DDY_select").find(".select button").text(text);
  $(this).closest(".option").removeClass("active");
  $(".DDY_select .select").removeClass("active");
  $(this).closest(".DDY_select").find(".select").addClass("font_c");
  $(this).closest(".DDY_select").find(".select").removeClass("error");
  if ($(this).parent().hasClass("disable")) {
    $(this)
      .closest(".DDY_select")
      .find(".select")
      .removeClass("font_c")
      .find("button")
      .text("전체");
  }
});

//영역밖 클릭시 remove
$(document).on("click", function (e) {
  if ($(e.target).closest(".DDY_select").length > 0) {
    return;
  }
  $(".DDY_select .option").removeClass("active");
  $(".DDY_select .select").removeClass("active");
});
