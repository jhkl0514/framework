    • Layout (레이아웃)
    • Tab (탭)
    Design Code

    Tab

    • 동일 계층의 콘텐츠를 그룹화해 화면 이동 없이 빠르게 여러 개의 콘텐츠 섹션을 전환할 수 있는 컴포넌트이다.

    구조

    • 1탭 :

      사용자와의 상호작용에 반응하는 박스 영역으로 실행 시 각 탭에 상응하는 콘텐츠가 탭 패널에 표시됨

    • 2레이블 :

      표시할 탭 항목

    • 3선택된 탭 인디케이터 :

      탭이 선택된 상태임을 알려주는 시각적인 식별자

    타입

    Underlined형

    • PC,Mobile 모두 공통으로 사용하며, 항목의 수가 많아질 경우 좌우 scroll로 항목을 이동한다.

    Outlined형

    • PC에서만 사용하며, 항목의 너비는 동일한 비율로 적용한다.

    Round형

    • Mobile에서만 사용하며, 2depth에서 사용하며 항목의 수가 많아질 경우 좌우 scroll로 항목을 이동한다.

    사이즈 및 간격

    Underlined형

    항목 간의 패딩 값을 유지하고, 항목 너비는 텍스트 길이에 따라 유동적으로 적용한다.

    Outlined형

    한 화면 내 탭이 모두 노출되도록 사용하며, 탭 항목의 너비는 동일한 비율로 적용한다.

    Round형

    항목 간의 패딩 값을 유지하고, 항목 너비는 텍스트 길이에 따라 유동적으로 적용한다.

    사용법

    • • PC에서 두 개의 탭을 중첩으로 사용할 경우

    • Outlined형이 1순위가 되며 대분류, Underlined 형이 소분류로 사용한다.

    • • Mobile에서 두 개의 탭을 중첩으로 사용할 경우

    • Underlined 형은 1depth로 사용하고, Round형은 2depth로 사용한다.

    • • 상태값에 따른 탭 형태를 사용하도록 한다.

    • Underlined형에서는 Hover값을 가지며, Outlined형에서는 앞 뒤 탭에서는 라운드 값을 가집니다.

    • • 탭에 사용 불가 상태를 사용하지 않는다.

    • 탭에 표시될 콘텐츠가 없는 경우, 해당 탭을 삭제하거나 탭을 선택했을 때 패널에 콘텐츠가 없는 이유에 대한 설명을 제공하여 사용자가 혼동을 느끼지 않도록 해야 한다.

$(".tabTop span").on("click", function () {
  const idx = $(this).index();
  $(".tabTop span").removeClass("active");
  $(this).addClass("active");
  $(".tabCon").hide();
  $(".tabCon").eq(idx).show();

  //tabTop span 클릭시 tableCon 조정 09_DDYlist
  $(".tabCon").children(".tableTabTop").find("span").removeClass("active");
  $(".tabCon")
    .eq(idx)
    .children(".tableTabTop")
    .find("span")
    .eq(0)
    .addClass("active");

  $(".tabCon").eq(idx).children(".tableCon").hide();
  $(".tabCon").eq(idx).children(".tableCon").eq(0).show();

  //searchBox
  $(".searchBox").removeClass("active");
});
