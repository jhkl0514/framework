Design Code

Calendar

• 달력은 날짜와 관련된 정보와 기능을 제공하는 데 사용한다.

구조

  • 1연/월 :

    달력의 연도, 월 정보 달력 유형에 따라 연도나 월을 선택하기 위한 컨트롤이 제공될 수 있음

  • 2연/월 탐색 버튼 :

    이전 또는 다음 연도, 월을 탐색하기 위한 버튼

  • 3요일 :

    한 주의 요일 정보

  • 4활성화되지 않은 날짜 :

    이전 달 또는 다음 달에 속한 날짜

  • 5날짜 :

    가장 기본적인 상태의 날짜(이번 달) 정보로 따라 버튼으로 사용될 수 있음

  • 6선택된 날짜(선택) :

    사용자가 선택한 날짜로 다른 상태의 날짜와 구분하기 위한 별도의 식별자와 함께 사용됨

  • 7호버 상태 날짜 :

    날짜를 호버 했을때 다른 상태의 날짜와 구분하기 위해 사용됨

타입

날짜 선택형

• 특정 일자를 선택하기 위해 사용하는 유형

스케줄형

• 일자별 설치/회수 스케쥴을 확인하기 위해 사용하는 유형

사용법

날짜 표기

• 기본적으로 연월일 및 요일은 YYYY-MM-DD (W)으로 표기한다.

• 서술형 문장에는 숫자 뒤에 년, 월, 일을 기재하며, 월과 일이 한자리 숫자인 경우 앞에 0을 표기하지 않는다. (ex. 오늘은 2023년 5월 5월입니다.)

• [1개월],[+30일]은 시작 날짜로부터 전 달 동일 날짜의 전날까지를 의미한다. (ex. 23.03.15~23.02.14)

• [이번달]은 이번달 1일부터 말일까지를 의미한다. (ex. 23.04.01~23.04.30)

디바이스별 날짜 선택

• PC에서는 Dropdown 형태를 활용하여 날짜를 선택한다.

• 모바일에서는 바텀시트를 활용하여 날짜를 선택한다.

디바이스별 월 선택

• PC에서는 Dropdown 형태를 활용하여 월을 선택한다.

• 모바일에서는 바텀시트를 활용하여 날짜를 선택한다.

• 모바일에서 “전체" 항목이 포함되어야 하는 경우 리스트 형태를 사용한다.

디바이스별 년도 선택

• PC에서는 Dropdown 형태를 활용하여 월을 선택한다.

• 모바일에서는 바텀시트를 활용하여 날짜를 선택한다.

function onToday() {
  const picker = tui.DatePicker.createRangePicker({
    language: "ko custom",
    startpicker: {
      date: today,
      input: "#startpicker-input",
      container: "#startpicker-container",
    },
    endpicker: {
      date: today,
      input: "#endpicker-input",
      container: "#endpicker-container",
    },
  });
}

function onYesterday() {
  const yesterday = new Date(today);
  yesterday.setDate(today.getDate() - 1);

  const picker = tui.DatePicker.createRangePicker({
    language: "ko custom",
    startpicker: {
      date: yesterday,
      input: "#startpicker-input",
      container: "#startpicker-container",
    },
    endpicker: {
      date: yesterday,
      input: "#endpicker-input",
      container: "#endpicker-container",
    },
  });
}

function onWeek() {
  const week = new Date(today);
  week.setDate(today.getDate() - 7);

  const picker = tui.DatePicker.createRangePicker({
    language: "ko custom",
    startpicker: {
      date: week,
      input: "#startpicker-input",
      container: "#startpicker-container",
    },
    endpicker: {
      date: today,
      input: "#endpicker-input",
      container: "#endpicker-container",
    },
  });
}

function onMonth() {
  const firstDay = new Date(today.getFullYear(), today.getMonth(), 1);

  const picker = tui.DatePicker.createRangePicker({
    language: "ko custom",
    startpicker: {
      date: firstDay,
      input: "#startpicker-input",
      container: "#startpicker-container",
    },
    endpicker: {
      date: today,
      input: "#endpicker-input",
      container: "#endpicker-container",
    },
  });
}

function onLastMonth() {
  const firstDay = new Date(today.getFullYear(), today.getMonth() - 1, 1);
  const lastDay = new Date(today.getFullYear(), today.getMonth(), 0);

  const picker = tui.DatePicker.createRangePicker({
    language: "ko custom",
    startpicker: {
      date: firstDay,
      input: "#startpicker-input",
      container: "#startpicker-container",
    },
    endpicker: {
      date: lastDay,
      input: "#endpicker-input",
      container: "#endpicker-container",
    },
  });
}

function onThreeMonth() {
  const lastMonth = new Date(today);
  lastMonth.setMonth(today.getMonth() + -3);
  const picker = tui.DatePicker.createRangePicker({
    language: "ko custom",
    startpicker: {
      date: lastMonth,
      input: "#startpicker-input",
      container: "#startpicker-container",
    },
    endpicker: {
      date: today,
      input: "#endpicker-input",
      container: "#endpicker-container",
    },
  });
}

function onLastYear() {
  const lastMonth = new Date(today);
  lastMonth.setFullYear(today.getFullYear() + -1);
  const picker = tui.DatePicker.createRangePicker({
    language: "ko custom",
    startpicker: {
      date: lastMonth,
      input: "#startpicker-input",
      container: "#startpicker-container",
    },
    endpicker: {
      date: today,
      input: "#endpicker-input",
      container: "#endpicker-container",
    },
  });
}
