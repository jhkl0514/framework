    Badge (배지)

    • Components
    • Layout (레이아웃)
    • Badge (배지)
    Design Code

    Badge

    • 컴포넌트에 대한 빠른 인지와 탐색을 돕기 위해 컴포넌트 근처에 표시되는 작은 문자 또는 숫자 데이터이다.

    구조

    • 1Dot :

      새로운 업데이트가 있을 경우 점으로 표현하는 유형

    • 2Container :

      배지를 배경과 구분하는 윤곽선

    • 3Label :

      배지를 통해 전달하고자 하는 데이터

    타입

    Dot

    • 아이콘과 함께 사용한다.

    • 텍스트 레이블 성격에는 사용하지 않는다.

    숫자형

    • 서비스 항목의 업데이트 수, 새로운 알림 등을 숫자로 카운팅하여 노출하는 유형

    • 아이콘과 함께 사용한다. 텍스트와 함께 사용하지 않는다.

    텍스트형

    • 데이터의 상태 정보를 제공하고자 할 경우 텍스트로 노출하는 유형

    • 아이콘과 함께 사용하지 않는다.

    사이즈 및 간격

    사용법

    • • 배지의 색상은 지정된 색상으로만 사용한다.

    • 배지 레이블이 전달하는 정보가 특정 색상이 가진 일반적인 의미 체계와 일치하는 경우에 맞춰 색상을 사용한다. 색상은 지정된 색상 이외의 색은 사용하지 않는다.

    • • 숫자형일 경우 최대 노출 표시 숫자는 99+까지로 한정한다.

    • 컨텐츠 확인 시 배지는 확인 처리되어 사라진다. 컨텐츠 미확인 시 배지는 계속 노출 상태를 유지한다.
      컨텐츠 미확인 시 배지 노출 기간에 대한 정의는 각 서비스 정책에서 정의한다.

    • • 한가지 종류의 배지만 사용한다.

    • 배지를 조합하여 두가지 이상의 종류는 함께 사용하지 않도록 한다. (텍스트형+Dot, 텍스트형+숫자 사용 불가)
