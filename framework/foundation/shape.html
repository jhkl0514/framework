Shape (형태)

  • Foundation
  • Shape (형태)

Corner radius

• 아이덴티티를 표현할 수 있는 Shape의 둥글기를 설정한다.

• Radius 값을 최대 10가지 정도로 구성한 후 각 컴포넌트 크기에 따라 대응하여 사용한다.

Radius px 설명
radius0 0px radius 값이 없는 상태
radius1 4px 라벨, 버튼. 텍스트 필드. 셀렉 필드 등 기본 Radius 값
radius2 8px 툴팁, 토스트, 모바일 컨텐츠 박스 등의 컴포넌트에 사용한다.
radius3 16px 모달 컴포넌트에 사용한다.
radius4 40px 바텀시트 상단에 사용되며 컨텐츠와 구별한다.
radius-circle 999px 원형의 모양에서 사용되는 값

Elevation

• 화면 안에서 일어나는 구성 요소들의 상,중,하를 표현하기 위한 방법으로, 시각적으로 위계를 구분하여 사용자가 채널 서비스 이용을 집중 할 수 있도록 유도한다.

shadow Value 설명
shadow1 x,y,blur(0,2,16) #000000,0.08
x,y,blur(0,0,1) #000000,0.2 		페이지 박스 형태에서 그림자 값
shadow2 x,y,blur(0,1,2) #000000,0.12
x,y,blur(0,0,1) #000000,0.2 		툴팁에서 사용하는 그림자 값 (단계차이가 작은 그림자)
shadow3 x,y,blur(0,2,4) #000000,0.12
x,y,blur(0,0,2) #000000,0.2 		셀렉트박스 옵션그룹,토스트 알림에서 사용하는 그림자 값
shadow4 x,y,blur(0,2,4) #000000,0.12
x,y,blur(0,0,2) #000000,0.2 		팝업,테이블(슬라이드 형태)에서 사용하는 그림자 값
shadow5 x,y,blur(-8,0,16)#000000,0.12
x,y,blur(0,0,8) #000000,0.2 		Left형 그림자 값 (PC-지도 리스트 분할 화면 사용)
shadow6 x,y,blur(0,-4,16)#000000,0.12 Top형 그림자 값 (모바일-지도 리스트 분할 화면 사용)

Border

• 테두리는 3 가지(1px,1.5px, 2px) 형태를 제공한다.

• borde-2(1.5px)은 페이지 단위에서는 사용을 지양하며, 아이콘에서만 사용할 것을 권장한다.

shadow px 설명
border1 1px 가장 일반적인 기본 값으로 사용한다.
border2 1.5px 아이콘에서만 사용할 것을 권장한다.
border3 2px 시각적으로 컴포넌트가 선택이 되거나 강조가 필요한 특수한 케이스에 사용한다.
