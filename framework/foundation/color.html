    Color (색상)

    • Foundation
    • Color (색상)

    Color

    • 웹 콘텐츠 접근성 지침(WCAG)의 AA 레벨을 충족하는 색상별 10단계(50,900 제외)를 지원하며, Grayscale을 제외한 색상별 주요 색상은 600이다.

    • Color Palette는 가장 밝은 색상을 50, 가장 어두운 색상을 900으로 한다.

    • Color는 명확한 커뮤니케이션을 위해 색상 명을 사용한다.

    ✅ 독자적인 Brand color를 Primary color로 사용하는 기관의 경우 Key color 규칙 적용을 권고

    • 명암비가 웹 콘 텐츠 접근성 지침(WCAG 2.1)의 AA 레벨을 충족해야 하며, 색상별 10단계(50, 900 제외)를 지원해야 한다.

    Brand color

    • Brand color는 버튼, 활성 상태와 같은 UI 전체의 주요 구성 요소에 사용된다.

    brandSub color

    brandSub Color는 필터, 칩과 같이 UI에서 보조적인 역할을 하는 구성 요소 전반에 사용된다.

    neutral

    neutral Color는 정보전달과 위계를 나타내는 중요한 역할을 한다. 주로 BG, Text, line에 사용되며 특히 텍스트는 가독성을 고려해야 하기 때문에 배경과의 명암비를 고려해야 한다.

    Usage

    States

    • 상태 표시 색상은 상황별 의미를 전달하는 색상으로 4가지 유형으로 사용하며, 기준 색상(Primary color)에서 점진적으로 증가하도록 정의한다. 편의성을 위해 Hover/Activated 같은 색상을 적용.

    Background

    배경색상으로 사용한다.

    State Color

    • UI요소들의 기능 또는 정보 유형에 따라 의미를 강조하여 전달하는 등의 목적으로 보조적으로 사용하는 컬러 (라벨에서 사용 가능)

    Danger

    • 오류, 금지, 위험, 삭제, 불가 등을 표시할 때 사용한다.

    Warning

    • 실수나 오류가 발생하지 않도록 주의 사항을 전달할 때 사용한다.

    Success

    • 성공, 완료, 진행 등의 결과를 전달할 때 사용한다.

    Information

    • 정보를 제공하거나 특별히 강조가 필요할 때 사용한다.
