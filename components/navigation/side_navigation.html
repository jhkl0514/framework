Design Code

Side Navigation

• 사이드 메뉴는 서브 화면 내에서의 이동을 위해 사용하는 메뉴이다.

• 일반적으로 본문 영역의 좌측에 사이드바 형태로 제공된다.

구조

• 사이드 메뉴의 가장 기본적인 형태로 링크가 적용되며, 사용자는 메인 메뉴와 메인 메뉴의 식별자를 통해 사이드 메뉴 목록의 계층 구조를 파악할 수 있다.

  • 1헤더(1depth)

    메뉴 목록의 제목을 제공하는 영역

    a. 제목메뉴 목록 전체의 상위 수준 정보 구조를 알려주는 텍스트

    b. 화살표제목으로 제공되는 상위 수준 정보 구조로 이동하는 데 사용되는 버튼

  • 2기본 화면 링크 목록 (2depth)

    기본 탐색 링크 그룹

  • 3하위 수준 페이지 링크 목록 (3depth)

    기본 페이지 링크 하위 수준에 속하는 링크 목록

  • 4꺽쇠 아이콘

    하위 수준 화면 링크 목록의 확장/축소 상태를 나타냄. 사용자는 꺽쇠 아이콘의 유무를 통해 개별 기본 화면 링크 하위에 추가적으로 탐색할 수 있는 링크 목록이 존재하는지를 인지할 수 있음

    a. 축소상위 목록

    b. 확장하위 목록

  • 5선택

    메뉴 목록 선택

유형

• 기본 : 2depth까지 펼쳐짐

• 기본 : 3depth 기본형태는 2depth 접혀있는 상태에서 방향 화살표 오른쪽 상태

상태

• 메뉴 목록은 기본적으로 모두 펼쳐져 있으며, 메뉴 목록을 열고 닫을 수 있다.

• 메뉴 선택 시 해당 메뉴와 관련된 페이지로 이동한다.

  • 1Unselcted-Default

    기본 메뉴 상태

  • 2Unselcted-Hover:Active:Press

    기본 메뉴 상태 - 선택

  • 3Selcted-Default

    선택 상태 - 기본 메뉴

  • 4Selcted-Hover:Active:Press

    선택 상태 - 선택

사이즈 & 간격

• 메뉴 목록은 기본적으로 모두 펼쳐져 있으며, 메뉴 목록을 열고 닫을 수 있다.

• 메뉴 전체 간격은 위/아래 : 24px, 좌/우 : 16px, depth gap : 4px

• 메뉴 높이는 1depth : 42px, 2/3depth : 40px
