Design Code

Toast

간단한 알림 메시지로 즉각적이고 일시적인 피드백을 제공한다.

구조

  • 1컨데이너 :

    알림이 표시되는 영역

  • 2아이콘 :

    알림상태를 나타내는 아이콘 (디폴트 : 체크, 오류:X)

  • 3텍스트 :

    전달하고 하는 알림 메시지 노출

  • 4설명 텍스트(옵션) :

    부가적인 설명 텍스트

  • 5액션버튼(옵션) :

    액션이 필요한 경우 액션버튼 노출

  • 6닫기 (옵션)

    토스트 메시지를 사용자가 직접 닫을 수 있는 버튼

타입

Single LIne

• 한 줄을 기본 값으로 가지며 최대 글자수는 공백포함 “36자"(PC 기준), “16자(MO)”입니다. 두 줄을 넘어가지 않도록 한다.

Single LIne with Action

• 부가적인 설명이 필요할 때 사용되며 메인 텍스트, 디스크립션 모두 한 줄을 기본값으로 가진다.

• Action 버튼과 닫기 버튼은 상황에 맞게 비노출 할 수 있다.

색상

사이즈 및 간격

사용법

  • • 토스트 메세지는 노출 2초 후 사라지도록 설정한다.

  • 단, 서비스에 따라 노출 시간을 다르게 설정할 수 있다. 액션버튼과 닫기 버튼이 노출 되는 경우는 자동으로 사라지는 것이 아닌 사용자가 직접 제어하도록 한다.

  • • PC, 모바일 모두 같은 컴포넌트를 사용한다.

  • PC에서는 가로값: 520px 고정값을 가지며, 모바일은 가로 화면의 80%를 사용한다.

  • • 메시지는 화면 가운데 하단에 정렬하고, Z값의 최상위에 노출한다.

  • PC는 하단 기준 48px. 모바일은 28px 위에 노출 한다. 또한 토스트는 버튼을 가리지 않도록 배치한다.

  • • 토스트 메시지가 2개 이상 노출 될 때에는 겹치지 않도록 한다.

  • 토스트들 사이의 간격은 PC는 16px, 모바일은 12px로 설정한다.

function Alert() {
  const previewLength = $("#preview_list > div").length;
  if (previewLength == 0) {
    $(".alert").addClass("on");
    let licustom = $(`<p class="error">
    <img src="/framework/assets/images/part_alarm_check.svg" alt="메세지" />메세지</p>`);
    $(".alert").append(licustom);
    setTimeout(function () {
      $(licustom).remove();
      $(".alert").removeClass("on");
    }, 2000);
  }
}
