Design Code

Tooltip

컴포넌트 주변에 배치되어 해당 컴포넌트의 상태나 관련된 상세 정보를 제공하는 컴포넌트이다. 정보 아이콘이나 도움 아이콘 버튼을 통해 사용자가 요청하는 경우에만 화면에 표시된다.

구조

  • 1아이콘 버튼 :

    도움말 콘텐츠의 표시하는버튼. 정보의 유형에 따라 정보(i) 아이콘, 물음표(?) 아이콘으로 사용

  • 2컨테이너 :

    도움말 영역과 배경을 구분하는 요소

  • 3제목(옵션) :

    도움말의 제목

  • 4본문 :

    구체적인 설명 텍스트

  • 5닫기 (옵션) :

    자동으로 닫히지 않은 경우, x 아이콘 버튼을 통해 닫음

타입

박스형
말풍선형
바텀시트형

• 모바일에서는 시각적 집중도를 위해 바텀 시트를 사용하여 정보를 전달한다.

색상

상태

  • 1하단으로 노출 :

    도움말 아이콘 아래쪽으로 노출됨

  • 2상단으로 노출 :

    도움말 아이콘 위쪽으로 노출됨

  • 3오른쪽으로 노출 :

    도움말 아이콘 오른쪽으로 노출됨

  • 4왼쪽으로 노출 :

    도움말 아이콘 왼쪽으로 노출됨

사이즈 및 간격

사용법

  • • 툴팁의 자동 닫침 시간은 2초로 설정한다.

  • 닫기 버튼이 없는 툴팁의 경우 2초 후 자동으로 툴팁이 닫히도록 설정한다.

  • • 텍스트는 간결하게 사용하도록 한다.

  • 제목 텍스트는 최대 한줄, 본문텍스트는 최대 3줄을 넘어가지 않도록 한다.

  • • 아이콘 버튼은 도움 정보를 제공하고자 하는 요소 주변에 배치한다.

  • 정보를 제공하고자 하는 컴포넌트나 레이블 옆에 배치하여 관련 요소와의 연관성이 명확하게 인지될 수 있도록 한다.

  • • 팝오버 영역이 본문의 중요 콘텐츠를 가리지 않도록 표현한다.

  • 팝오버 영역이 활성화 버튼과 맥락적으로 도움을 제공하고자 하는 본문 콘텐츠 요소를 가리지 않도록 적절한 표시 방향을 설정한다.
