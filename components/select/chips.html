Design Code

Chips

• 칩은 사용자로 하여금 정보, 선택, 콘텐츠의 필터링, 그리고 특정 액션을 유도한다.

구조

  • 1 텍스트 레이블 :

    칩을 눌렀을 때 실행되는 기능에 대한 설명

  • 2컨테이너(선택) :

    칩을 배경과 구분하는 시각적인 수단으로 면 또는 선으로 표현됨

  • 3아이콘(선택) :

    첩을 눌렀을 때 실행되는 기능을 텍스트 대신 시각적인 상징으로 표현한 것

Option chips

• 사용자가 여러 옵션 중 하나를 선택하도록 하는 데 사용한다.

• Active 시 즉각적으로 반영된 액션과 변화가 저장되며 완료 버튼을 누르기 전까지 유지된다.

• 사용자가 옵션 칩을 클릭하면 선택되고 해당 옵션이 활성화한다. (단일 선택, 클릭하면 선택됨)

• 칩 중 하나를 선택 시, 변화가 없는 경우가 Basic Option Chips이다.

• 카테고리 선택, 설정 변경 등에서 활용가능하다.

Filter chips

• 사용자가 콘텐츠를 필터링하도록 하는 데 사용한다.

• 사용자가 필터 칩을 선택하면 해당 필터링 조건이 즉각적으로 반영된 액션과 테이터가 나타난다.

• 다중 선택 가능하다.

• 검색 결과 필터링, 상품 목록 정렬 등에서 활용가능하다.
