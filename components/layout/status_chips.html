Design Code

Status Chips

• Status Chip은 주문완료, 출고대기, 주문완료와 같이 상태가 변하는 값에 사용하는 값이다.

구조

  • 1컨데이너 :

    배경과 구분하는 윤곽선 또는 배경색

  • 2아이콘 :

    레이블과 관련된 아이콘

  • 3레이블 :

    상태칩을 통해 전달하고자 하는 데이터

사이즈&간격

사용법

  • • 상태/ 스타일에 따라 구분하여 사용

  • 상태값은 6개(Info, Success, Error, Default, BrandSub, Caution), 스타일은 3개(Filled, Transparent, Outline)로 구분되어 있다.
    상태값에 맞는 상태칩을 사용하는 것을 권장한다.

  • • 레이블 앞 아이콘 비노출 가능

  • 레이블 앞 아이콘의 경우 필요에 따라 비노출 처리하여 텍스트 형태로만 사용가능하다.

  • • 상태 칩은 하나의 영역 안에서 한개만 사용 가능

  • 현재 상태를 나타내는 값을 표현하는 칩이므로 하나의 영역안에서는 한개 칩만 사용이 가능하다. 두개 이상을 노출하고자 할때는 배지를 활용하여 표현하는 방법이 있다.
