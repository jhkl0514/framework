Label (레이블)

  • Components
  • Layout (레이아웃)
  • Label (레이블)
Design Code

Label

• 관련 구성요소(컴포넌트)에 대한 설명 텍스트를 제공하여 사용자 이해를 돕는 컴포넌트

구조

  • 1Label :

    레이블을 통해 전달하고자 하는 내용

  • 2Component :

    레이블과 연관된 컴포넌트 노출

타입

기본형

• 기본형 레이블은 연관된 양식 필드가 선택 사항이며 제출에 필수가 아님을 나타내는 데 사용된다.

필수형

• 기본형 레이블은 연관된 양식 필드가 필수 사항( * )임을 나타는데 사용한다.

사용법

  • • 레이블은 컴포넌트와 조합하여 사용한다.

  • 텍스트 필드, 셀렉트 필드,탭 등 정의된 컴포넌트와 조합하여 사용하며, 그 외 경우 레이블을 사용하지 않고 타이틀처럼 별도로 사용한다.

기본형
